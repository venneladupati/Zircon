
Welcome to GoLearn's codebase (previously named Zircon).
GoLearn is a tool designed to augment lectures at the University of Minnesota.
GoLearn can summarize, generate notes, and render short-form video content hereafter referenced as "brainrot".
View the demonstration below to see GoLearn in action.

![GoLearn Demo](/frontend/public/demo.mp4)

# Codebase Overview
The codebase is split into four parts, backend, extension, frontend, and terraform. Each section has their own `README.md` describing how to operate them and their dependencies.
